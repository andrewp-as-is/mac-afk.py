### Installation
```bash
$ [sudo] pip install mac-afk
```

```bash
$ [sudo] npm i -g mac-afk
```

#### Examples
```python
>>> import mac_afk
>>> import time
>>> time.sleep(2); mac_afk.seconds()
2
```

```bash
$ sleep 3 && afk
3
```
